# NEXA HACK Submission - SemesterOS

SemesterOS is a design-first student operating system that transforms chaotic semester obligations into a beautiful, actionable command center.

## What it does
- Captures course tasks, deadlines, exams, clubs, and personal goals.
- Visualizes semester pressure using timeline + heatmap + priority queue.
- Generates concise AI plans for realistic weekly execution.
- Includes focus mode and milestone tracker for momentum.

## Why it stands out
- Distinctive cinematic UI, glassmorphism cards, and high-contrast gradients.
- End-to-end narrative flow optimized for memorable first 30 seconds.
- Creative “student OS” framing instead of a generic task manager.
